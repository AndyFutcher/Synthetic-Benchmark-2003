Tweaking SynthMark XP

 

See Also: Advanced Task Manager, Back to Main.

 

In this chapter we discuss a few common methods of improving your performance on SynthMark XP, to make even a slower computer competitive with more powerful ones. In the previous 4 chapters we fully describe why SynthMark XP is the easiest benchmarking system to tweak, because its synthetic. And this makes any form of optimization easier and more effective, as it is with your applications and programs running on your computer.

 

Just a reminder: This whole section you do at your own risk!

 

General Tweaking:

 

·          Scandisk & defragment your computer often.

·          Do not move your mouse while your computer benchmarks, unless asked to.

·          If you are running Windows 2000, XP or NT try increasing SynthMark XP’s priority in the task manager.

·          Use the advanced task manager to close unwanted or unneeded background services.

·          Keep you SynthMark XP window maximized for better performance.

·          Keep your content managers settings low to minimize excess memory usage, especially if you have low memory.

·          Update your computers drivers; see DriverMAGIC Pro at our website.

 

Windows XP Tweaking:

 

·          Disable system restore, and visual effects for low memory computers, also disable remote assistance, automatic updating. All these can be done on your system properties window found under Start >> Control Panel >> System.

·          Another thing you can do it set your Virtual memory swap file size to a static one, by making your maximum and minimum sizes the same keeps your swap file the same size. You can do this under Start >> Control Panel >> System >> Advanced >> Settings (performance) >> Advanced >> Change.

·          With Windows XP, you can prevent certain services from loading on start-up, do this through Start >> Control Panel >> Administrative Tools >> Services.

 

Hardware Tweaking:

 

·          Upgrade your hardware, or sometimes adding more RAM can big a big difference too.

·          Changing certain BIOS options can improve performance. (see below).

·          Hardware Overclocking can improve performance significantly, but you should research it online at many overclockers sites before attempting it.

 

How to increase SynthMark XP Priority:

 

·          Hold the CTRL-ALT-DEL key simultaneously.

·          Click the processes tab and you will see a list of open services.

·          Select the one in the list called synthmark_xp.exe and right-click on it.

·          Now you will see a menu open with the option set priority at the bottom.

·          Under that option, you will see a few options, Lowest to Real-time.

·          NOTE: Use the ‘realtime’ option only if you have a DUEL PROCESSOR SMP computer, if you have only one processor your system will crash at that level.

·          For single processor users you can choose the ‘High’ option for best results.

·          Note that the higher you set the priority the more the processors represent your hardware, and the lower you set the option the more the score represent the operating systems services efficiency.
