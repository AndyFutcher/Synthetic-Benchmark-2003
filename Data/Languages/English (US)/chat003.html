Calling Commands & Voting with SynthMark XP.

 

See Also: Kicking a Platform, Adding or Removing URLs, Back to Main.

 

One of the most useful features when it comes to SynthMark XP chatting is the ability for you to call commands. We can add any commands at a later stage, if you have any suggestions please email them to Our Submissions Manager.

 

We currently support the following commands; these commands are called either by clicking the button called ‘command’ next to the ‘say’ button. Or by entering in a front slash at the beginning of the say work box, followed by whatever command is supported.

 

Command

Example

Description

Kick a Platform

/kick Jimmy Nimrod

This will remove jimmy nimrod from this server.

Ban a Platform

/ban Jimmy Nimrod

This will remove jimmy nimrod from this server and prevent him from entering next time. The ban will last only this session, not forever.

Add a URL

/addhttp www.AndyFutcher.com

This will add a url to the servers URL benchmark list for the Web processing benchmark.

Remove a URL

/remhttp www.AndyFutcher.com

This will remove the desired URL from the server’s benchmark list, if it exists.

Vote Yes

/yes

If a vote is in progress, entering this command will call your vote as a YES.

Vote No

/no

If a vote is in progress, this command will call your vote as a NO.

 

Additional Notes on Voting:

 

·          You may forfeit a vote at any time, the YES votes must exceed the NO votes to win.

·          If you are the host computer, any of the above commands will instantly do what it calls.

·          If you are a client computer, any of the above commands will instantly call a vote before it can be done.

·          You cannot kick or ban the server computer.

·          With the remove URL option, opening your configuration window, double-clicking any of the URLS will call a remove vote for that URL.
