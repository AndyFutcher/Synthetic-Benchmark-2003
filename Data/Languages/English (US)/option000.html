SynthMark XP Options & Special Features!

 

Go To: Back to Main.

 

SynthMark XP is loaded with useful features and flexible options for almost every aspect of the program. Generally the default for all these options is what most users will want or need, but under certain situations you may need to change an option or two to get the desired effect, or even setup a connection type, ect. Here is a list of all the main options and features and a guide on how or when to use them.

 

 

Topics:

The Bad Language filter

If you want to filter out all the nasty words, this option is for you.

Check for Connectivity on Start-up

If you are connected to the internet whenever you boot up, then this feature will save you a few seconds.

Enable Update Detection

Do you want to know when there are new updates available for your SynthMark XP?

Network Connection Speed

Choose which connection type best describes your connection type for reliable server access.

Save file Compression Level

Especially if you save large project files, changing the levels will balance speed over size.

Connection Types & Proxy Setup

If you are running a specific connection type (and SynthMark XP has not detected it correctly).

Checking for SynthMark XP Updates

How to manually check for SynthMark XP updates, this includes bad language and CPU support updates.

Using the Burn-in feature

Want to test the reliability of your computer? The burn-in feature is the only way to do it.

Using the Advanced Task Manager*

Optimizing your computer is all part of the SynthMark XP deal, the ATM allows you to close background services and tweak your computer.

Using the Password Tool (protection)*

If paranoia is a big thing in your mind, you can password protect any projects you save, making them secure to editing.

Editing your User Profile

Changing your name and/or you user comments, a few suggestions on using the comments section.

Using the Content Manager

This manager helps you define how much data is stored in SynthMark XP’s memory reserves and file format.

* full version features
