Loading & Saving Presets with SynthMark XP.

 

See Also: Choosing your Benchmarks, Back to Main.

 

This chapter is quite simple and discusses the way you can save your current benchmark configuration for use at a later stage. The menus also have some common presets for your to select, and you may get quite used to using them because sometimes your can battle to keep track of all the benchmarks you have selected.

 

Load & Save Presets Menu Options:

Load Preset File

Loads a .sxpreset file that can only be made by saving a benchmarking preset with the save option.

Save Preset File

Saves your current benchmark selection into a file with the extension .sxpreset. For use at a later stage.

Select Typical

Select all the common benchmarks, the verbose benchmarks and the multi-media benchmarks. (stays open)

Select None

This option will automatically deselect all the benchmarks under SynthMark XP. (stays open)

Select All

This option will select all the benchmarks it can select with SynthMark XP. (stays open)
