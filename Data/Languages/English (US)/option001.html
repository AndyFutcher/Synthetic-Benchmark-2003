Using the Bad Language Filter.

 

See Also: Enable Update Detection, Kicking & Banning a Platform, Back to Main.

 

Sometimes you can get quite unruly characters on internet chat rooms, so to help improve the situation we made a bad language filter to try and make sure that if other people want to inflict their poor vocabulary on you, they can’t do it completely. This also give you time to call a vote and get them kicked out before they can do it again.

 

Please people, respect the integrity of the other benchmarkers out there and don’t get yourself to the point where you have to use bad language. We will be filling our minds with all the nastiest words in the world to try and keep other users from using them on each other.

 

Using the bad Language Filter:

 

1.       Once SynthMark XP is open, any time click the ‘Tools’ drop down menu.

2.       Followed by the Options menu at the bottom of the list.

3.       You will then see an options window appear.

4.       The first the option right at the top says ‘Always use the bad language filter’.

5.       Make sure it is enabled and you should have no swearing displayed to you.

6.       If you find that some words get through, send them to us and we will update it for the good of everyone using SynthMark XP.

 

The bad language filter will replace all bad words found with a series of zeros with lines through them. [ ø ] If you disable the bad language filter you will be able to see all the swearwords you want. But then you have to wonder “who are these people you are mixing with” on the web. This is a free world, we leave that for you to decide?

 

Clicking the update button will check to see if there are newer versions of the bad language filter available online.
