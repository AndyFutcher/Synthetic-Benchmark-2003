SynthMark XP Connection Types & Proxy Setup.

 

See Also: Check for Connectivity on Start-up, Back to Main.

 

One of the most important configurations for SynthMark XP is its internet connectivity setup. Although most users would never need to change this setup, there are cases were a user might need to modify on of the settings because SynthMark XP may have incorrectly detected the current internet connection.

 

Even though these cases are few and far between we still have to tell you how do change the setups and to know when to use what type of connection. The main reason for this configuration is the setup of a proxy server on your local network, if SynthMark XP did not detect your proxy connection properly, you will need your system administrator to give you the correct proxy address (just like a IP address) and port number for internet access.

 

Connection Type Summary:

 

Connection Type

Description

Auto Detect Connection Type

Lets SynthMark XP automatically detect what connection you are running.

Direct or Modem Access

If you are using any kind of dial up connections like a modem, cable or ISDN connection.

Via a Proxy Server

If you are running SynthMark XP off a big network, it is most likely you are running a proxy server internet access.

 

How to Change the Connection Type:

 

1.       Once Synthmark XP is open, click the ‘Tools’ drop down menu.

2.       Followed by the options menu.

3.       This will open the SynthMark XP main options window.

4.       The section at the bottom of the options window will have the options like the ones shown below.

5.       You will not need to enter any addresses or port numbers unless you select the ‘Via a Proxy Server’ option type.

6.       The other connection types automatically configure themselves.
