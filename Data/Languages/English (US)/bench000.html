Benchmarking with SynthMark XP

 

See Also: Managing Benchmarks, Using NETMark, Back to Main.

 

Well, SynthMark XP is a benchmarking system and so it should actually be able to benchmark. This trick is not that it can benchmark, but rather that there are so many benchmarks to perform, some very serious, others very new and some are quite fun too! One of SynthMark XP big thumbs up is that it has so many benchmarks to perform, all this with NETMark to keep the scores relevant and the fun, constant!

 

SynthMark XP is all you will ever need to satisfy your morbid benchmarking desires and at the same time waste some punks evil box with higher specs than yours! Anyway let us not dwell on warm feeling before we have an accident, and get to explaining exactly what you need to know about benchmarking with SynthMark XP.

 

 

Topics:

Choosing your Benchmarks

The question is, to benchmark, or not to benchmark? We will help you decide that.

Configuring the selected Benchmarks

Some benchmarks are configurable, and some have to be configured. Quite simple really.

Central processing benchmarks (CPU)

This benchmark group specially targets your processors, chipset, cache and memory performance.

Floating point benchmarks (FPU)

Operating system benchmarks

Are you running Windows™ or are you running Windoze? Find out now!

Hard Drive & Cd-Rom benchmarks*

Multi-Media benchmarks*

Movies, music and more, the multi-media benchmarks are for all you media editors out there.

The Web processing benchmark*

Network & Internet benchmarks*

If you are a client platform, you can test your connection speed with various communication benchmarks.

Interactive Benchmarks (Fun)
