Permanent Limitations:

The ATM Advanced Mode

The advanced task manager (ATM) normal view mode works and allows you to close certain tasking running on your computer, however in advanced mode you close even hidden tasks like the following:

 

·          Hidden Fax Window

·          Connections Tray Program

·          Print Spoolers

·          Power Connection Trays

·          And hundreds more!

 

All this helps to save your computers processor time-slices and free up extra memory to improve you SynthMark XP scores, system performance is also improved but at the cost of less features.

 

The Password Protect Tool

Using this tool will set a password for any benchmark project files you save. These files normally have the extension (.sxp) and are used to save your projects in a highly compressed format. This has the following advantages:

 

·          You can’t by accident open that project and edit it.

·          Nobody else can open it and edit either.

·          Great for securely sending your projects to a friend without anyone else seeing them.

·          When submitting your projects to AndyFutcher, we suggest giving them a password and including it with the email.

·          This is the ultimate benchmarker paranoia fix!

Real-time Benchmarking Mode

This is the most powerful benchmarking mode for almost all the benchmarks, instead of balancing with your operating system & services on your system, SynthMark XP essentially crashes itself for a second at a time to get the highest possible application scores. You will find the real-time benchmarking mode is less dependant on a good, clean operating system and is more consistent with score. Full version users have this luxury.
