Using the Keyboard with SynthMark XP.

 

See Also: Differences in the Menus, Using the Explorer Panel, Back to Main.

 

Generally using SynthMark XP with the keyboard is like using any other Windows based program with the keyboard, we have taken the time to describe a few of the movements here and also some of the differences too.

 

 

The Key

Alternate Key / Info

Description

TAB

 

Cycles through the visible selectable objects in the current window.

ENTER

SPACE BAR

Acts like a click, or click of a default button for the current window in the case of a text box.

ESC

 

Click the cancel button for the current window.

Cursor Keys ßàáâ

 

Moves to the next similar control, eg the next menu or check box.

ALT

Left or right

Jumps to any object with a single underlined character in its title.

Front Slash or /

 

Used in SynthMark XP to begin a command.

 

 

One of the biggest differences in SynthMark XP is the ALT key is no longer used primarily for the drop down menus. It is used to jump to any key with an underlined character in its caption or title. This is great for quick navigation and means if you ever press the ALT key by accident, it wont jam up the other controls while you are supposed to be searching through the menus.

 

Generally you will always have a better experience using a mouse and using the ALT key quickly to navigate if you are busy typing with the chat box and don’t want to reach across to your mouse and then click the button.
