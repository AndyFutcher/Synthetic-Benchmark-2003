The SynthMark XP Status Window.

 

See Also: Dedicated Servers, Back to Main.

 

The SynthMark XP NETMark status window allows you to see exactly what connections SynthMark XP is hosting. This window is especially useful when you are running a dedicated server.

 

The status window show you two main things, firstly your connection status to the internet, either to the Andy Futcher NETMark server, or two the internet in general. And secondly any direct connections to your computer, these direct connections only appear when you host a server or connect to another server.

 

The Status Windows Glossary:

Unknown

This means that SynthMark XP has not checked that connection type yet.

Online

This means that the current connection type is available at the moment.

Offline

This means that the services provided by this connection type are not available presently. You either have to connect to the internet (dial up) or enable your connection hardware.

Waiting…

This connection number is waiting for a user to join through this connection.

Connected

A computer is connected to this connection number.

 

The disconnect button is used to close all the connections if you are a client or a server computer and connected using the NETMark system.
