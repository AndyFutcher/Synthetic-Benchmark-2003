Using the Content Manager with SynthMark XP.

 

See Also: Using the Password Tool, Managing your Benchmarks, Back to Main.

 

The content manager defines for SynthMark XP how much of everything should be stored in it memory and in its saved .SXP files format. Remember the more you add on to each section the more memory SynthMark XP will use for browsing and convenience purposes and the less that will be available for the actual benchmarks. So if you have a low spec. computer we suggest keeping these settings as low as you can.

 

Server or dedicated servers should also use restraint with these settings as large quantities of text will be transfers every time someone joins and you have the chat catch-up system running.

 

How to open the Content Storage Window:

 

1.       Open SynthMark XP.

2.       You don’t have to open a benchmark session to change these options.

3.       Click the ‘Tools’ drop down menu followed by the Content Manager option.

4.       A window like the one below will be displayed and you will be able to setup your storage.

5.       Remember that the values have minimum value as well.

 

 

You can use this window anytime, even if you are in the middle of chatting with a few friends online or benchmarking offline. The effect is taken the next benchmark you run or in the chats case the next line of chat added.
