Specifying a Server with SynthMark XP.

 

See Also: SynthMark XP and Firewalls, Back to Main.

 

In the case of the server you are locally trying to connect to uses a different port number, you may have to specify this port number, and if you know the other computers IP or friendly name (computer name) you can connect to the other computer right there and then.

 

·          A computers IP is a four sets of numbers with a maximum value of 255, 0 being a number. And a common IP address will look something like this: 192.168.0.81.

·          A computers friendly name can be determined with Windows 98 or ME under the control panel >> network properties, or with Windows 2000 or XP under the control panel >> system >> computer name (tab).

 

Notes on connecting computers with SynthMark XP:

SynthMark XP determines which address is your network address by looking at your subnet mask. If your subnet mask is full this means that that IP is your internet connection, and if it is half full it is a Local area networking IP. SynthMark XP also shows the friendly name of the current computer and it’s found IP addresses in the start-up window.

 

You can use any port number you want, as long as they are both the same and unblocked by a firewall if you have one, the computers will connect seamlessly.
